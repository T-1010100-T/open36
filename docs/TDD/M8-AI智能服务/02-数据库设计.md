## 1. 数据库概述

### 1.1 数据库实例

M8服务使用独立的数据库 `ai_db`，与现有服务共用同一个PostgreSQL实例（开发环境），生产环境可独立部署。

### 1.2 数据实体

| 实体 | 说明 | 核心字段 |
|------|------|---------|
| `ai_conversations` | 对话会话 | id, user_id, title |
| `ai_messages` | 对话消息 | id, conversation_id, role, content |
| `ai_tasks` | 异步任务 | id, user_id, task_type, status, result |
| `ai_tool_calls` | 工具调用记录 | id, message_id, tool_name, input, output |
| `ai_agent_configs` | Agent配置 | id, agent_name, model, system_prompt |

### 1.3 ER关系

```
ai_conversations 1──N ai_messages
ai_messages 1──N ai_tool_calls
ai_tasks (独立表，关联user_id)
ai_agent_configs (独立配置表)
```

## 2. 表结构设计

### 2.1 ai_conversations - 对话会话表

管理员与Agent的对话会话。

| 字段名 | 类型 | 约束 | 说明 |
|--------|------|------|------|
| `id` | UUID | PK | 会话ID |
| `user_id` | BIGINT | NOT NULL | 管理员用户ID（关联M1） |
| `title` | VARCHAR(200) | NOT NULL | 会话标题（自动从首条消息提取） |
| `status` | VARCHAR(20) | NOT NULL, DEFAULT 'active' | 状态：active/archived |
| `created_at` | TIMESTAMP | NOT NULL, DEFAULT NOW() | 创建时间 |
| `updated_at` | TIMESTAMP | NOT NULL, DEFAULT NOW() | 更新时间 |

**状态说明**：

- `active`：活跃会话，可继续对话
- `archived`：已归档，不再接收新消息

### 2.2 ai_messages - 对话消息表

每条对话消息，包含用户输入和Agent回复。

| 字段名 | 类型 | 约束 | 说明 |
|--------|------|------|------|
| `id` | UUID | PK | 消息ID |
| `conversation_id` | UUID | FK, NOT NULL | 关联会话ID |
| `role` | VARCHAR(20) | NOT NULL | 角色：user/assistant/system |
| `content` | TEXT | NOT NULL | 消息内容 |
| `intent` | VARCHAR(50) | NULL | 识别的意图（仅assistant消息） |
| `agent_name` | VARCHAR(50) | NULL | 处理该消息的Agent名称 |
| `token_usage` | JSONB | NULL | Token使用量 `{"input": N, "output": N}` |
| `created_at` | TIMESTAMP | NOT NULL, DEFAULT NOW() | 创建时间 |

**role说明**：

- `user`：管理员输入
- `assistant`：Agent回复
- `system`：系统消息（如错误通知）

### 2.3 ai_tasks - 异步任务表

异步执行的任务记录。

| 字段名 | 类型 | 约束 | 说明 |
|--------|------|------|------|
| `id` | UUID | PK | 任务ID |
| `user_id` | BIGINT | NOT NULL | 创建者用户ID |
| `conversation_id` | UUID | FK, NULL | 关联会话（可选） |
| `task_type` | VARCHAR(50) | NOT NULL | 任务类型 |
| `status` | VARCHAR(20) | NOT NULL, DEFAULT 'pending' | 任务状态 |
| `input_data` | JSONB | NOT NULL | 任务输入参数 |
| `result_data` | JSONB | NULL | 任务执行结果 |
| `error_message` | TEXT | NULL | 错误信息（失败时） |
| `agent_name` | VARCHAR(50) | NOT NULL | 执行的Agent名称 |
| `retry_count` | INT | NOT NULL, DEFAULT 0 | 重试次数 |
| `max_retries` | INT | NOT NULL, DEFAULT 3 | 最大重试次数 |
| `started_at` | TIMESTAMP | NULL | 开始执行时间 |
| `completed_at` | TIMESTAMP | NULL | 完成时间 |
| `created_at` | TIMESTAMP | NOT NULL, DEFAULT NOW() | 创建时间 |
| `updated_at` | TIMESTAMP | NOT NULL, DEFAULT NOW() | 更新时间 |

**task_type枚举**：

| 值 | 说明 | 典型耗时 |
|----|------|---------|
| `single_post` | 单篇发帖 | 10-30s |
| `batch_post` | 批量发帖 | 30s-5min |
| `single_problem` | 单题生成 | 20-60s |
| `batch_problem` | 批量出题 | 1min-10min |

**status枚举**：

| 值 | 说明 |
|----|------|
| `pending` | 等待执行 |
| `running` | 执行中 |
| `completed` | 已完成 |
| `failed` | 执行失败 |
| `cancelled` | 已取消 |

### 2.4 ai_tool_calls - 工具调用记录表

Agent执行过程中对工具（后端API）的调用记录，用于审计和调试。

| 字段名 | 类型 | 约束 | 说明 |
|--------|------|------|------|
| `id` | UUID | PK | 调用记录ID |
| `message_id` | UUID | FK, NULL | 关联消息（同步调用时） |
| `task_id` | UUID | FK, NULL | 关联任务（异步调用时） |
| `tool_name` | VARCHAR(100) | NOT NULL | 工具名称 |
| `tool_input` | JSONB | NOT NULL | 工具输入参数 |
| `tool_output` | JSONB | NULL | 工具输出结果 |
| `status` | VARCHAR(20) | NOT NULL, DEFAULT 'pending' | 调用状态 |
| `error_message` | TEXT | NULL | 错误信息 |
| `duration_ms` | INT | NULL | 执行耗时（毫秒） |
| `created_at` | TIMESTAMP | NOT NULL, DEFAULT NOW() | 创建时间 |

**status枚举**：

- `pending`：调用中
- `success`：成功
- `failed`：失败

### 2.5 ai_agent_configs - Agent配置表

各Agent的配置信息，支持动态调整。

| 字段名 | 类型 | 约束 | 说明 |
|--------|------|------|------|
| `id` | UUID | PK | 配置ID |
| `agent_name` | VARCHAR(50) | UNIQUE, NOT NULL | Agent名称 |
| `display_name` | VARCHAR(100) | NOT NULL | 显示名称 |
| `model` | VARCHAR(100) | NOT NULL | 使用的LLM模型 |
| `system_prompt` | TEXT | NOT NULL | 系统提示词 |
| `temperature` | DECIMAL(3,2) | NOT NULL, DEFAULT 0.7 | 温度参数 |
| `max_tokens` | INT | NOT NULL, DEFAULT 4096 | 最大输出Token |
| `is_enabled` | BOOLEAN | NOT NULL, DEFAULT true | 是否启用 |
| `created_at` | TIMESTAMP | NOT NULL, DEFAULT NOW() | 创建时间 |
| `updated_at` | TIMESTAMP | NOT NULL, DEFAULT NOW() | 更新时间 |

**预置Agent配置**：

## 3. 索引设计

### 3.1 索引列表

```sql
-- ai_conversations
CREATE INDEX idx_conversations_user_id ON ai_conversations(user_id);
CREATE INDEX idx_conversations_status ON ai_conversations(status);
CREATE INDEX idx_conversations_created_at ON ai_conversations(created_at DESC);

-- ai_messages
CREATE INDEX idx_messages_conversation_id ON ai_messages(conversation_id);
CREATE INDEX idx_messages_created_at ON ai_messages(created_at);

-- ai_tasks
CREATE INDEX idx_tasks_user_id ON ai_tasks(user_id);
CREATE INDEX idx_tasks_status ON ai_tasks(status);
CREATE INDEX idx_tasks_task_type ON ai_tasks(task_type);
CREATE INDEX idx_tasks_created_at ON ai_tasks(created_at DESC);
CREATE INDEX idx_tasks_user_status ON ai_tasks(user_id, status);

-- ai_tool_calls
CREATE INDEX idx_tool_calls_message_id ON ai_tool_calls(message_id);
CREATE INDEX idx_tool_calls_task_id ON ai_tool_calls(task_id);
CREATE INDEX idx_tool_calls_created_at ON ai_tool_calls(created_at);

-- ai_agent_configs
CREATE UNIQUE INDEX idx_agent_configs_name ON ai_agent_configs(agent_name);
```

### 3.2 索引说明

| 表 | 索引 | 用途 |
|----|------|------|
| `ai_conversations` | `user_id` | 查询用户的会话列表 |
| `ai_tasks` | `user_id, status` | 查询用户特定状态的任务 |
| `ai_tasks` | `status` | Worker查询pending任务 |
| `ai_tool_calls` | `task_id` | 查询任务的工具调用记录 |

## 4. SQL建表语句

```sql
-- M8 AI智能服务 数据库初始化脚本
-- 数据库: ai_db

-- 会话表
CREATE TABLE ai_conversations (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    user_id BIGINT NOT NULL,
    title VARCHAR(200) NOT NULL,
    status VARCHAR(20) NOT NULL DEFAULT 'active',
    created_at TIMESTAMP NOT NULL DEFAULT NOW(),
    updated_at TIMESTAMP NOT NULL DEFAULT NOW()
);

-- 消息表
CREATE TABLE ai_messages (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    conversation_id UUID NOT NULL REFERENCES ai_conversations(id) ON DELETE CASCADE,
    role VARCHAR(20) NOT NULL CHECK (role IN ('user', 'assistant', 'system')),
    content TEXT NOT NULL,
    intent VARCHAR(50),
    agent_name VARCHAR(50),
    token_usage JSONB,
    created_at TIMESTAMP NOT NULL DEFAULT NOW()
);

-- 异步任务表
CREATE TABLE ai_tasks (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    user_id BIGINT NOT NULL,
    conversation_id UUID REFERENCES ai_conversations(id) ON DELETE SET NULL,
    task_type VARCHAR(50) NOT NULL,
    status VARCHAR(20) NOT NULL DEFAULT 'pending' CHECK (status IN ('pending', 'running', 'completed', 'failed', 'cancelled')),
    input_data JSONB NOT NULL,
    result_data JSONB,
    error_message TEXT,
    agent_name VARCHAR(50) NOT NULL,
    retry_count INT NOT NULL DEFAULT 0,
    max_retries INT NOT NULL DEFAULT 3,
    started_at TIMESTAMP,
    completed_at TIMESTAMP,
    created_at TIMESTAMP NOT NULL DEFAULT NOW(),
    updated_at TIMESTAMP NOT NULL DEFAULT NOW()
);

-- 工具调用记录表
CREATE TABLE ai_tool_calls (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    message_id UUID REFERENCES ai_messages(id) ON DELETE CASCADE,
    task_id UUID REFERENCES ai_tasks(id) ON DELETE CASCADE,
    tool_name VARCHAR(100) NOT NULL,
    tool_input JSONB NOT NULL,
    tool_output JSONB,
    status VARCHAR(20) NOT NULL DEFAULT 'pending' CHECK (status IN ('pending', 'success', 'failed')),
    error_message TEXT,
    duration_ms INT,
    created_at TIMESTAMP NOT NULL DEFAULT NOW()
);

-- Agent配置表
CREATE TABLE ai_agent_configs (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    agent_name VARCHAR(50) UNIQUE NOT NULL,
    display_name VARCHAR(100) NOT NULL,
    model VARCHAR(100) NOT NULL,

);

CREATE INDEX idx_conversations_created_at ON ai_conversations(created_at DESC);
CREATE INDEX idx_messages_conversation_id ON ai_messages(conversation_id);
CREATE INDEX idx_messages_created_at ON ai_messages(created_at);
CREATE INDEX idx_tasks_user_id ON ai_tasks(user_id);
CREATE INDEX idx_tasks_status ON ai_tasks(status);
CREATE INDEX idx_tasks_task_type ON ai_tasks(task_type);
CREATE INDEX idx_tasks_created_at ON ai_tasks(created_at DESC);
CREATE INDEX idx_tasks_user_status ON ai_tasks(user_id, status);
CREATE INDEX idx_tool_calls_message_id ON ai_tool_calls(message_id);
CREATE INDEX idx_tool_calls_task_id ON ai_tool_calls(task_id);
CREATE INDEX idx_tool_calls_created_at ON ai_tool_calls(created_at);
CREATE UNIQUE INDEX idx_agent_configs_name ON ai_agent_configs(agent_name);
```
